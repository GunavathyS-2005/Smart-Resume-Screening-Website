# Smart Resume Screening Website

## Project Overview

Smart Resume Screening Website is a web application designed to simplify and automate the resume screening process.

The system allows recruiters to upload multiple resumes and compare them with a job description. Based on skills, experience, keyword matching, and resume quality, candidates are analyzed and ranked automatically.

This helps reduce manual effort and improves recruitment efficiency.

---

## Problem Statement

Recruiters often receive a large number of resumes for a job role.

Manually checking every resume is time-consuming and may lead to missing suitable candidates.

This project attempts to solve this issue by automating the resume analysis and ranking process.

---

## Features

* Upload multiple resumes
* Enter a job description
* Extract skills and important information
* Compare resumes with job requirements
* Display matched and missing skills
* Generate candidate score
* Rank candidates automatically

## Core Logic

The project works using:

### Keyword Matching

The system extracts important skills and keywords from resumes and compares them with the job description.

### NLP-Based Scoring

The system analyzes relevance between resumes and job requirements and generates a score based on matching criteria.
